- **Primary Purple**: #9333ea (rgb(147, 51, 234))

## Icons Used

- 🎫 Ticket - Event/Seats
- 📅 Calendar - Date
- 🕐 Clock - Time
- 📍 MapPin - Location
- 👥 Users - Attendees
- 💳 CreditCard - Payment
- ← ArrowLeft - Back navigation
- ⏳ Loader - Processing state

## States

### Normal State
- All fields editable
- Confirm button enabled (if form valid)
- Purple color scheme

### Loading State
- Confirm button shows spinner
- Button text: "Processing..."
- Button disabled
- Prevents double submission

### Error State
- Red toast notification
- Form remains editable
- User can retry

### Success State
- Green toast notification
- Redirect to dashboard
- Booking appears in "My Bookings"
